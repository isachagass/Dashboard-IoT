Este projeto foi desenvolvido como uma tarefa de classe, com o objetivo de simular o controle de uma casa inteligente por meio de um dashboard interativo.

A aplicação foi construída utilizando React, explorando a criação de componentes reutilizáveis e a organização da interface em um painel centralizado, permitindo a visualização e o controle de diferentes funcionalidades de uma residência inteligente. O dashboard simula ações como o acionamento de dispositivos, fornecendo uma experiência próxima à de sistemas reais de automação residencial.

O foco do projeto foi o desenvolvimento de uma interface dinâmica e intuitiva, utilizando recursos do React para atualização de estado em tempo real e melhor experiência do usuário. A proposta permitiu aplicar conceitos fundamentais de front-end, como componentização, gerenciamento de estados e estruturação de layouts para dashboards.

Este projeto representa uma prática introdutória no desenvolvimento de interfaces interativas para sistemas de IoT, reforçando conhecimentos em React e construção de painéis de controle visuais.
